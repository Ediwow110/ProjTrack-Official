# Evidence Reports

This directory is for local evidence reports generated by:

```bash
npm run evidence:local
```

The command writes files named like:

```text
local-evidence-YYYY-MM-DDTHH-MM-SS-sssZ.md
```

## Rules

Generated reports are not proof by themselves. They count only after the relevant results are copied or summarized into the official evidence documents:

- `docs/CI_STATUS.md`
- `docs/SECURITY_ACCEPTANCE_GATE.md`
- `docs/2ND_MAIN_IMPROVEMENTS.md`
- the relevant GitHub evidence issue

## When to commit a generated report

Commit a generated report only if it is intentionally being used as a durable audit artifact for a specific commit SHA.

If a report contains environment-sensitive paths, hostnames, usernames, tokens, or other sensitive details, do not commit it. Instead, summarize the safe parts in the evidence documents.
