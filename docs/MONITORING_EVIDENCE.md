## Required Alerts

- API unavailable for 5 minutes.
- Readiness failure for 5 minutes.
- Database unreachable.
- Mail worker heartbeat stale.
- Backup worker heartbeat stale.
- Backup failure or missing recent backup.
- High 5xx rate.
- Authentication failure spike.
- File upload rejection spike.
